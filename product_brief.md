# Product Brief: CSV Dedupe, Normalize, And Schema Cleanup API

## Target User

People who repeatedly receive messy CSV exports from forms, CRMs, ecommerce tools, scraped tables, spreadsheets, or client files and need a clean, deduplicated file quickly.

## Problem

CSV cleanup is common but annoying:

- headers are inconsistent;
- cells contain extra spaces;
- blank rows pollute imports;
- duplicate records appear after merges;
- rows sometimes have missing or overflow columns;
- people need a report explaining what changed.

## MVP Promise

Upload or submit a CSV and receive:

1. a cleaned CSV;
2. normalized column names;
3. duplicate-row removal;
4. row-shape repair;
5. a JSON cleanup report.

## First Publish Path

RapidAPI is the likely first marketplace because the utility is naturally API-shaped and can charge for rows/files processed.
